# Module 2 Activity - Temperature Conversion

## Purpose & Concepts
The purpose of this activity is to practice previously covered concepts and the following new concepts: Primitive data types, Expressions, Modulo, Widening/Narrowing conversions, Automatic assignment conversion, Casting, Promotion

## Problem Description
In this activity you will create a new class called TemperatureConversion.java that will prompt the user for a temperature in Fahrenheit, calculate the corresponding temperature and display it in a meaningful way to the user. You will clone the example repository to help you in this.
 
## Requirements
1. Store the temperature in Fahrenheit in a variable. You will convert this value.
2. Use mathematical expressions to convert the Fahrenheit value to Celsius. Store this in another variable.
3. Display the converted temperature in a meaningful way. (e.g. "75 degrees Fahrenheit is equivalent to 23.9 in Celsius")
